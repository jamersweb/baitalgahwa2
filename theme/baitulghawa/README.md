# Baitulghawa Moodle Theme

Baitulghawa is a custom Moodle theme built as a child theme of Boost for Moodle 5.2+.

## Install

1. Copy this folder to `moodle/theme/baitulghawa`.
2. Open Moodle as an administrator.
3. Go to `Site administration > Appearance > Themes > Theme selector`.
4. Select `Baitulghawa`.
5. Purge caches from `Site administration > Development > Purge caches`.

## Customisation

The theme exposes:

- Brand colour
- Preset selector
- Raw initial SCSS
- Raw SCSS

Most visual work lives in `scss/pre.scss` and `scss/post.scss`.
